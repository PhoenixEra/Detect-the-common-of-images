各函数说明：

1  main：主函数，包括读入图片，计算每张图片的自相似性描述子表示，计算显著性得分，画出检测结果；


2  com_Self_Similarities：用于计算图片的自相似性描述子，该部分包括如下一些子函数：

2.1  cal_ssd：用于计算patch与region的SSD；

2.2  get_self_sim_vec：用于将将2.1计算得到的相似性转换为自相似性描述子向量；

2.3  cart2polar：用于将直角坐标转换为以区域中心为原点的极坐标系；

2.4  get_bin：根据2.3得到的极坐标系，用于将极坐标系划分为15*3=45个bin（论文中采用80bin，但是处理太慢，这里把图像进行了缩放，因此相应减少了bin的划分）

（注：为了加快计算，事先通过2.3和2.4将bin直接划分好，之后类似查表就行）


3  draw_result：用于画出共同部分的检测结果。
